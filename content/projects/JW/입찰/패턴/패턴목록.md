# 패턴 목록 마스터

> 모든 패턴의 ID·상태·적중 이력을 관리한다.
> 판단 노트에서 패턴 ID로 참조. 개찰 후 반드시 이 파일을 업데이트한다.

---

## 사용법

- **ID**: P001, P002 … 순서대로 부여
- **분류**: 규칙성 / 반복성 / 경향성
- **상태**: 활성 / 소멸 / 승급검토
- **확인 횟수**: 반례 없이 작동한 횟수 (PRE 판단 시 근거로 삼은 횟수가 아니라, POST 적중 횟수)
- **반례**: 불발된 횟수
- **승급 기준**: 반복성 → 규칙성은 확인 4회 이상 + 반례 0일 때 검토

---

## 패턴 목록

| ID | 패턴 요약 | 분류 | 상태 | 확인 횟수 | 반례 | 도출값/범위 | 최초 등록 | 최종 갱신 |
|----|-----------|------|------|-----------|------|-------------|-----------|-----------|
| P001 | (예시) 3월~5월 바닥값 XX%대 수렴 | 반복성 | 활성 | 2 | 0 | XX.X~XX.X% | 2026-01-00 | 2026-06-00 |

---

## 소멸 패턴 아카이브

> 반례 누적 또는 경향성 교체로 더 이상 유효하지 않은 패턴. 삭제하지 않고 보관.

| ID | 패턴 요약 | 분류 | 소멸 판단 근거 | 소멸 확인일 |

|    |           |      |                |             |
